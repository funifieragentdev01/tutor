# Character Style Guide

## Reference
- See `character-reference-beatriz.jpg`

## Style Description
- **Genre**: Chibi / Kawaii cartoon
- **Proportions**: Big head (~40% of body), small body, cute/childlike
- **Eyes**: Large, expressive, anime-influenced with highlights
- **Face**: Rosy cheeks, simple nose, big smile
- **Outlines**: Clean, slightly thick, darker than fill color
- **Colors**: Flat with soft gradient shading, pastel-friendly
- **Shading**: Minimal, soft — no hard shadows
- **Background**: Transparent or simple pattern (polka dots)
- **Pose**: Friendly, waving or standing — approachable
- **Clothing**: Matches real photo — same outfit, same colors

## AI Prompt Template (for generating from photo)
"Create a chibi kawaii cartoon character based on this photo. Big head, small body, large expressive anime eyes with highlights, rosy cheeks, clean outlines, flat colors with soft shading. Match the exact clothing, hair color, hair style, and skin tone from the photo. Friendly standing pose, waving hand. Transparent background. Full body view."
